Mint Development Environment 1

Este ambiente foi criado para servir como uma máquina virtual de desenvolvimento pronta para uso em laboratório.

O sistema operacional escolhido foi o Linux Mint, utilizando o XFCE como Desktop Environment. O principal motivo dessa escolha é performance: XFCE possui baixo consumo de memória e CPU, o que é ideal para máquinas virtuais rodando em computadores de laboratório.

Além do XFCE, foi incluído também o Window Manager i3 como opção no login. O i3 permite organizar janelas automaticamente na tela (tiling) e utilizar o computador principalmente através do teclado, o que pode aumentar bastante a produtividade para quem prefere esse tipo de fluxo de trabalho.

Outro ponto importante é que esta máquina virtual foi configurada para ser efêmera. Isso significa que todas as alterações feitas durante o uso são descartadas quando a máquina virtual é desligada. Ao iniciar novamente, o sistema volta ao estado original da imagem.

Isso evita problemas como:

  • arquivos esquecidos por alunos anteriores
  • configurações quebradas
  • credenciais salvas indevidamente
  • conflitos de ambiente entre usuários diferentes

Diferenças importantes

1. C#

A forma tradicional de se desenvolver aplicações em C# é utilizando o Visual Studio, que não possui suporte para Linux.

No Linux, o desenvolvimento normalmente é feito utilizando:

  • .NET CLI
  • Visual Studio Code
  • Extensões da Microsoft para C#

Vídeo de configuração:

https://www.youtube.com/watch?v=chsJ6W2tfJA

2. XAMPP

A ferramenta XAMPP não foi instalada.

O ambiente já possui:

  • Apache
  • PHP
  • MySQL / MariaDB

A pasta /xampp/htdocs foi criada apenas por conveniência.

Atalhos avançados: i3 Desktop Environment

O XFCE foi escolhido pela leveza da máquina virtual.

Para quem prefere trabalhar com teclado, recomenda-se utilizar o i3.

Documentação:

https://i3wm.org/docs/userguide.html
